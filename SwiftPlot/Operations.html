Syntax and Operations
Variables

The syntax of SwiftPlot is based on mathematical expressions. There's one big difference; "=" means "will be", instead of "equals". between all variables there must be an operator.
The first variable in a Model Line will be assigned the result of said Line.
v = v + a * dt
v + a * dt.
the variable v will be assigned the value of v + a * dt. The = sign is ignored. This allows for shortening the line above:
Note: SwiftPlot takes math order into consideration, hence it's important to take these into account.

Operations

SwiftPlot currently supports a limited amount of operations: Power, Multiply, Divide, Add and Substract.
Below you'll find what symbol represents what operation.
- Power: ^
- Multiply: *
- Divide: /
- Add: +
- Substract: -

if-then-else statements

The syntax for if-else-then statements is as follows:
if comparison then expression else expression
Example: if v >= 0 then Fres = Fengine - Ffriction else Fres = Fengine

Compare Operators

These compare operators are supported:
- equals: ==
- larger: >
- smaller: <
- larger and or equals: >=
- smaller and or equals: <=
Note: Due to mathematical limitations, equals currently behaves a little weird. For higher precision SwiftPlot calculates with Doubles, but that means that 5.00000, behind the scenes means 4.9999999999999.
