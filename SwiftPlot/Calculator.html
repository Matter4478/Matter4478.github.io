Calculator:

The calculator consists of two parts, the register and the calculator. The register is used to hold values and the calculator is where your instructions will take place.

Calculator:

The calculator uses the same syntax as the model feature: Syntax and Operations.
To hold a value in the register you simply type a letter or a name like so:
a = 2 * 3
This will store a = 6 in the register. To modify this value you can either write: a = a * 2 or write a * 2. Take note: the 'a =' is not required, it simply functions to make it clear to the user. In SwiftPlot this rule applys: The first variable will be assigned the result of the expression.*

Register:

The register bar can be hidden with the blue navigationbar button on the upperleft. The values being held by the register can be deleted by pressing the trashcan in the upperright corner.

* Technically speaking this means that as an example the value 2 can be assigned a different value. In the calculator this will be corrected just before the next calculation is handled.
