Variables are objects that have a name and a value. I'll try to explain this by example
The formula for speed is as follows: v = v + a * dt.
Here "v" stands for the value of the object named v.

Start Values

First press the check variables button. It's important to do this before assigning variables as they'll be reset.
All values will be assigned 0.0 by default. To change the start value you'll type a value into the textfield and click on the value to be changed. When all values are assigned the right values, it's safe to run the model.
Take note: The value of dt must not be nil, as this will cause the model to crash. If dt is too small or stop is too large the model will time out.
